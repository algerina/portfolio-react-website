## Getting Started

To get a local copy up and running follow these simple example steps.

```sh
git clone git@github.com:Algerina/portfolio-react-website.git 
```

Then 
```sh
cd portfolio-react-website
```

To install packages
```sh 
npm install
``` 

To Run the app
```sh 
npm start
```

To Run the test
```sh 
npm run test
```
